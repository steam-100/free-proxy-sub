# Free Proxy Subscription

Auto-updated free proxy list in Quantumult X and Clash format.

Updated every 30 minutes via GitHub Actions.

## Subscription URLs

Replace `{username}` with your GitHub username.

### Quantumult X

```
https://raw.githubusercontent.com/{username}/free-proxy-sub/main/output/qx.txt
```

### Clash

```
https://raw.githubusercontent.com/{username}/free-proxy-sub/main/output/clash.yaml
```
